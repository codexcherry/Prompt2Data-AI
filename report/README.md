# Report Generation - Prompt2Data AI

This folder contains scripts to generate evaluation plots and visualizations for the Prompt2Data AI project.

## Setup

1. **Install Python dependencies:**
   ```bash
   pip install -r ../requirements.txt
   ```

2. **Run the plot generator:**
   ```bash
   python generate_plots.py
   ```

## Generated Plots

The script generates 20 evaluation plots saved in the `images/` folder:

1. **System Architecture Diagram** - Overall system architecture
2. **Data Flow Diagram** - Data processing pipeline flow
3. **ML Model Performance Comparison** - Model performance metrics
4. **Feature Comparison Chart** - Feature capabilities comparison
5. **Technology Stack Visualization** - Tech stack breakdown
6. **API Endpoints Flow** - API endpoint workflow
7. **Model Training Time Comparison** - Training time metrics
8. **Task Type Detection Accuracy** - Auto-detection accuracy
9. **Data Processing Pipeline** - Step-by-step pipeline
10. **Supported File Formats** - File format support visualization
11. **Model Selection Matrix** - Model-task compatibility matrix
12. **Performance Metrics Dashboard** - Comprehensive metrics dashboard
13. **User Workflow Diagram** - User journey visualization
14. **Component Interaction Diagram** - Component relationships
15. **Error Handling Flow** - Error handling and recovery
16. **Data Quality Metrics** - Data quality assessment metrics
17. **Scalability Analysis** - Performance vs file size
18. **Feature Usage Statistics** - Feature adoption rates
19. **Security Features** - Security implementation status
20. **Project Statistics Summary** - Overall project statistics

## Output

All generated images are saved in `images/` folder with high resolution (300 DPI) for presentations and documentation.

## Customization

You can modify `generate_plots.py` to:
- Adjust colors and styles
- Add more plots
- Change data values
- Modify chart types
